Paths is a free, open source, browser-based daily word puzzle game where the central concept is the creation of a non-overlapping path through a grid of letters by spelling words. Each path is generated randomly with an algorithm that ensures the grid can be completely filled.

How To Play

Using the generated grid of letters, your goal is to create a non-overlapping "path" through the grid by spelling words. Words can be spelled forwards or backwards. For example, "BELL" or "LLEB". Each word must start or end on a letter adjacent to the last letter of the previous word (as displayed on the grid).

Letters can be selected by either tapping on them one-by-one or by pressing and then sliding across several letters. When selecting one-by-one, the selection can be submitted by re-tapping the final selected letter. When pressing and sliding across several letters, the selection will be submitted automatically when you stop pressing.

As you play, you may find it helpful to expand the drawer beneath the grid by tapping on the arrow (expand_more). This is where you can get more detailed information on your progress as well as where you can adjust game settings.

Paths has two game modes. By default, you will be playing in Pathfinder mode. You can change the game mode in the game settings section of the drawer.

Pathfinder Mode (Default)

Pathfinder is the default game mode. In this mode, the goal is to figure out the path used to generate the grid by spelling out each word that forms the path. If one of the words you spell contains a section of the path, that section of the path will be revealed on the grid by highlighting each matching cell.

For example, if the path word was "SUPERB" and you spelled "SUPER", it would highlight the five matching cells:

Challenge Mode

In Challenge mode, the goal is to try to get the highest score possible. While the grid is still generated in a way that ensures it can be fully filled, it may require swapping one or more letters in order to do so. Unlike in Pathfinder mode, there is no one set path, and there are no hints. You can create whatever path through the grid you like.

Secret Words

The words used to generate the grid are referred to in this mode as "secret" words. If you happen to spell one of these words, it will show up as a highlighted word in the list of spelled words and count towards the "Secret Words Found" statistic. If the path of the word on the grid is the same as the path used during grid generation, the word will also be highlighted on the grid itself, denoting that the placement of that word on the grid is optimal.

Swapping Letters

Letters can be swapped by tapping twice on a single letter, and then tapping on another letter to complete the swap. Swapped letters will have a purple background color. A single letter can only be swapped once. Swapped letters are excluded when calculating the score for a word, unless that word happens to be a "secret" word.

Scoring

A word will be scored after it is validated. To be considered valid, a word must be at least three letters long and must be contained in the dictionary. Validated words are awarded points based on the letters they contain as well as a bonus multiplier for meeting certain word length thresholds. Any swapped letters in a word are awarded 0 points and are excluded from word length calculations.
